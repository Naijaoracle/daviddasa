Tuberculosis Prediction

Chest x-rays are a very common imaging tool in healthcare. This means the ability to pick up tuberculosis from a chest x-ray is something that may be beneficial to patients. Eventually we may expand this model to identify other conditions from chest x-rays.

In the previous example here we used a convolutional neural network to classify x-rays into positive or negative for TB. It was based on the python library pytorch.

This model is built on the tensorflow library.

Conclusion

The project aims to achieve an accurate and reliable classification model for TB detection in chest X-ray images.

One of the previous models here had a high accuracy.

Next we will use a model which is pretrained for image classification to see if it can achieve a better performace at this task. see changes here.
