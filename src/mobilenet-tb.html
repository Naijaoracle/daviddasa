Tuberculosis Prediction

Chest x-rays are a very common imaging tool in healthcare. This means the ability to pick up tuberculosis from a chest x-ray is something that may be beneficial to patients.

In the previous examples, such as here we used a custom convolutional neural network to classify x-rays into positive or negative for TB.

This model is built on the pretrained MobileNet neural network, it was then trained on the data set.

This MobileNet work had a basis in the work of K. Scott Mader. You can read his notebook here.

Conclusion

The project aims to achieve an accurate and reliable classification model for TB detection in chest X-ray images.

The notebook above showcases the use of a convolutional neural network built on MobileNet. We can view it in comparison to the custom models here. This model had an accuracy of 88.89% on the testing data (with a validation loss of 0.29039).
