Featured Project

AI-Driven XR Simulation for Clinical Training

A research prototype integrating intelligent non-player characters (NPCs) into extended reality (XR) environments to enhance team-based clinical training. The main work involves AI-driven character colleagues running in virtual reality (e.g., Meta Quest headset) built with Unity and Unreal game engines using a cross-platform bridge. The system combines a real-time physiological model with AI agents capable of dynamic dialogue, guideline adherence, and adaptive teamwork support. Designed to evaluate how AI can improve realism, safety, and educational outcomes in high-stakes scenarios.

Current focus: Validating the DASEX framework for evaluating AI-driven simulations through pilot studies with healthcare learners.
