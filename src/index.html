Featured Project

Secure Patient Monitoring System

A project to build a remote patient monitoring system which uses short-range Bluetooth Low Energy medical devices, verifies proximity to a trusted device, and sends the data to medical professionals who verify the data is from a trusted device, saving it to a distributed ledger running a proof of authority consensus protocol.
